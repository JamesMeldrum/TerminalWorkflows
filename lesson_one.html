Terminal navigation:



  • cd - change directory
    cd /home/user-name/Documents
  • pwd - present working directory
    pwd
  • ls - list directory
    ls -al /home/user-name/Documents /bin
    -l - long list
    -lh - long list and show file size in a human-readable format
    -a - show hidden files

Terminal shortcuts:



  • EDIT: ctrl + alt + t - creates a new the terminal window
    ctrl + l - clears all text from your terminal
    ctrl + d - closes the terminal. EOF
    ~ - shorthand for your home directory
    cd ~ - go to your current home directory
    - - shorthand for your last directory
    cd - - go to the last directory you were in
    . - shorthand the current directory
    cd . - won't take you anywhere!
    .. - shorthand the parent directory
    cd .. - go to the parent of the current directory
  • pwd - present working directory
  • ls - list directory
    ls -al /home/user-name/Documents /bin
    -l - long list
    -lh - long list and show file size in a human-readable format
    -a - show hidden files

Core Utilities:



  • man - prints information about a utility
    man cat
    cat - displays the contents of a file
    cat /etc/motd
    echo - prints text to the terminal
    echo "hello world!"
    top - shows hardware usage of running processes
    ps - shows running processes for a user
    a - show all processes your computer is running
    au - show all processes your computer is running as well as who's run them
    less - view large files easier
    less /var/log/dmesg
    while less is open, pressing 'q' will quit, 'f' will move forward a page and 'b' will move back a page
    more - view large files easier
    more /var/log/dmesg
    while more is open, pressing 'q' will quit, 'f' will move forward a page and 'b' will move back a page

Package Management



Package management software available on many distributions of Linux can make installing and using new software much easier.
The two most popular package management systems are Aptitude for Debian-based (Ubuntu, Mint, etc) systems and yum for RHEL-based (Fedora, CENTOS) systems.
Please take this opportunity to install the following pieces of software if they are not already on your system:
  • git
  • vim
sudo apt-get install git vim or su yum install git vim

Editing Text with Vim



Vim is a case-sensitive text editor.
  • Standard Commands
    :w - save your document
    :q - quit Vim
    :wq - save and quit Vim
    i - leave normal mode and enter edit mode
    esc - leave edit mode and enter normal mode
  • Normal Mode
    arrow keys (or h, j, k,l) - cursor navigation v - highlight
    w - move forward a word
    b - move back a word
    y - copy selection
    yy - yank current line p - paste last yanked line u - undo ctrl + r - redo
    ctrl + n - keyword completion
    ctrl + f - Move forward a page forward
    ctrl + b - Move back a page forward
    G - Move to the end of a document
    gg - Move to the top of a document
    ctrl + w + vSplit the window vertically
    ctrl + w + sSplit the window horizontally
  • Configuration
    Vim is highly customizeable, usually through the .vimrc file located in your ho directory or through the use of Pathogen (available from GitHub).
    • Command Mode
    A number of commands are accessible from within Normal Mode by pressing ':'.
    Try some of the following. I'll let you work out what they do.:
    • :!ls
    • :!echo Hello World
    • :!uname -a
    • :set nu
    • :tabnew
    • :tabc
    • /search-term
    • /\ccase-insensitive-search-term
    • :%s/search/replace/g
    • :20,30s/limited-line-search/limited-line-replace/g
    • :set paste
