Introduction




I hear a lot that shell scripting is a dying art. Whilst it may be possible to do anything in shell in another language, it is hard to find a tool that will let you develop filesystem operations as quickly as a simple bash script. In Python, to view the contents of the home directory its an operation that requires specific calls on the os module and you'll need to hand-format the output after iterating the output. In bash, its 2 lines.


#!/bin/bash
# list.sh
echo `ls`

#!/env python
# list.py
import os
def list_dirs():
path = "/home/jrm"
dirList = os.listdir(path)
for fname in dirList:
print fname

if __name__ == "__main__":
list_dirs()


I wish to make clear that I'm not saying that bash is the highlander of programming languages, it has specific use cases that cannot be easily replicated.
